Правила оформления заголовков:
\begin{itemize}
  \item заголовки разделов и подразделов начинаются с абзацного отступа,
  \item в конце заголовка точка не ставится,
  \item переносы слов в заголовках не допускаются,
  \item аббревиатуры в заголовках не допускаются.
\end{itemize}

\section*{Примеры перечислений}
% -------------------------------------------------------

\textbf{Вариант 1 --- маркированный список через тире} (п.~4, Приложение~1 требований ИТМО):

Информационно-сервисная служба для обслуживания удалённых пользователей включает следующие модули:
\begin{itemize}
  \item удалённый заказ,
  \item виртуальная справочная служба,
  \item виртуальный читальный зал.
\end{itemize}

\textbf{Вариант 2 --- нумерованный список с цифрой и скобкой}:

Работа по оцифровке включала следующие технологические этапы:
\begin{enumerate}
  \item первичный осмотр и структурирование исходных материалов,
  \item сканирование документов,
  \item обработка и проверка полученных образов,
  \item выходной контроль качества.
\end{enumerate}

\textbf{Вариант 3 --- многоуровневый список}:

Разрабатываемое устройство можно применять в различных отраслях:
\begin{itemize}
  \item в машиностроении:
  \begin{enumerate}
    \item для очистки отливок от формовочной смеси,
    \item для холодной штамповки из листа;
  \end{enumerate}
  \item в ремонте техники:
  \begin{enumerate}
    \item устранение наслоений на внутренних стенках труб,
    \item очистка каналов небольшого диаметра.
  \end{enumerate}
\end{itemize}

\section*{Пример формулы}
% -------------------------------------------------------

Функция потерь определяется формулой~(\ref{eq:loss_app}):
\begin{equation}
L(y, \hat{y}) = \frac{1}{n} \sum_{i=1}^{n} (y_i - \hat{y}_i)^2,
\label{eq:loss_app}
\end{equation}
где $y_i$ --- истинное значение, $\hat{y}_i$ --- предсказанное значение, $n$ --- объём выборки.
