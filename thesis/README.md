# ITMO AITH Master Thesis Template

LaTeX-шаблон для магистерской диссертации AI Talent Hub. Шаблон соответствует актуальным [Требованиям к ВКР ИТМО](https://student.itmo.ru/files/1314) (версия 4.0, 2022 г., ГОСТ 7.32-2017).

Готовый шаблон для Overleaf: [открыть в Overleaf](https://www.overleaf.com/read/gxbcxjstynvj#90457e).

---

## Структура проекта

```
├── main.tex               % Основной файл — точка входа для компиляции
├── preamble.tex           % Все настройки LaTeX (поля, шрифты, отступы и т.д.)
├── abbreviations.tex      % Список сокращений и условных обозначений
├── terms.tex              % Список терминов и определений
├── bibliography.bib       % Список использованных источников
│
├── config-lists.tex       % Настройки списков (тире, нумерация)
├── config-formulas.tex    % Настройки формул и теорем
├── config-tables.tex      % Настройки таблиц
├── config-images.tex      % Настройки рисунков
│
├── intro.tex              % Введение
├── chapter1.tex           % Глава 1
├── chapter2.tex           % Глава 2
├── conclusion.tex         % Заключение
│
├── appA.tex               % Приложение А
└── example.jpg            % Пример с изображением
```

## Быстрый старт

### Вариант 1 — Overleaf (без установки)

1. Скачайте архив репозитория как `.zip`
2. В Overleaf: **New Project → Upload Project** → загрузите zip
3. **Обязательно смените компилятор** (шаблон использует `fontspec`, который требует XeLaTeX):
   - Нажмите **Menu** (кнопка в левом верхнем углу)
   - В разделе **Settings** → **Compiler** выберите **XeLaTeX**
   - Нажмите **Recompile**

### Вариант 2 — локально (VS Code или TeXstudio)

**Требования:**
- TeX Live (полная установка) или MikTeX
- XeLaTeX как компилятор (рекомендуется)
- Пакет `biblatex-gost` (входит в TeX Live)

**Компиляция через latexmk:**
```bash
latexmk -xelatex -f -interaction=nonstopmode main.tex
```

**Или вручную (4 прогона):**
```bash
xelatex main.tex
biber main
xelatex main.tex
xelatex main.tex
```

## Соответствие требованиям ИТМО

| Требование | Параметр | Реализация |
|---|---|---|
| Шрифт | Times New Roman, 14 пт | `\setmainfont{Times New Roman}`, `14pt` |
| Поля | лево 30 мм, право 15 мм, верх/низ 20 мм | `geometry` в `preamble.tex` |
| Межстрочный интервал | 1.5 | `\onehalfspacing` |
| Абзацный отступ | 1.25 см | `\setlength{\parindent}{1.25cm}` |
| Нумерация страниц | арабскими, по центру снизу, без точки | `fancyhdr`, `\cfoot{\thepage}` |
| Заголовки глав | прописными, полужирный, по центру | `\titleformat{\chapter}` + `\MakeUppercase` |
| Заголовки разделов | с абзацного отступа, полужирный | `\titlespacing{\section}{1.25cm}{...}` |
| Содержание | текст обычным шрифтом (не жирный) | `\cftchapfont{\normalfont}` |
| Подписи к рисункам | по центру, «Рисунок N — ...» | `captionsetup[figure]` |
| Подписи к таблицам | слева, «Таблица N — ...» | `captionsetup[table]` |
| Формулы | нумерация по главам: (1.1), (1.2) | `\numberwithin{equation}{chapter}` |
| Маркеры списков | тире | `label=--` в `enumitem` |
| Нумерованные списки | цифра со скобкой: 1), 2) | `label=\arabic*)` в `enumitem` |
| Ссылки на источники | в квадратных скобках `[1]` | автоматически через `biblatex` |
| Библиография | ГОСТ-numeric, по порядку цитирования | `style=gost-numeric, sorting=none` |
| Приложения | буквы русского алфавита (А, Б, ...) | `\cyrtoc` + переопределение `\appendix` |
| Сокращения/термины | алфавитный порядок, без знаков препинания | `abbreviations.tex`, `terms.tex` |
| Термины | без жирного шрифта | обычный текст в `terms.tex` |

## Как редактировать

### Структура ВКР (обязательные разделы по п. 3.2 требований ИТМО)

1. **Титульный лист** — оформляется в ИСУ ИТМО, не входит в этот шаблон
2. **Задание** — оформляется в ИСУ ИТМО
3. **Аннотация** — оформляется в ИСУ ИТМО
4. **Содержание** — генерируется автоматически в `main.tex`
5. **Список сокращений** → `abbreviations.tex`
6. **Список терминов** → `terms.tex`
7. **Введение** → `intro.tex`
8. **Основная часть** → `chapter1.tex`, `chapter2.tex`, ...
9. **Заключение** → `conclusion.tex`
10. **Список использованных источников** → `bibliography.bib`
11. **Приложения** → `appA.tex`, ...

### Нумерация страниц

По требованиям (п. 4.3.1) нумерация сквозная по всему тексту, начиная с листа «СОДЕРЖАНИЕ».

Титульный лист, задание и аннотация оформляются отдельно в ИСУ ИТМО и **не входят** в этот LaTeX-файл. Чтобы номер страницы на содержании совпал с фактическим положением листа в итоговом PDF (после объединения с титульными страницами), измените стартовый номер в `main.tex`:

```latex
% main.tex — измените число на нужное:
\setcounter{page}{4}  % если перед содержанием 3 страницы (титул, задание, аннотация)
```

### Добавление глав

1. Создайте файл `chapterN.tex`
2. Добавьте в `main.tex`:
```latex
\input{chapterN.tex}
```

### Рисунки

```latex
% Перед вставкой дайте ссылку в тексте: (рисунок~\ref{fig:example})
\begin{figure}[h]
    \centering
    \includegraphics[width=0.8\textwidth]{example.jpg}
    \caption{Описание рисунка}
    \label{fig:example}
\end{figure}
```

На все рисунки должны быть ссылки в тексте (п. 4.5.4).

### Таблицы

```latex
% Ссылка в тексте: (таблица~\ref{tab:example})
\begin{table}[h!]
    \caption{Название таблицы}
    \label{tab:example}
    \centering
    \begin{tabular}{|c|c|c|}
    \hline
    Столбец 1 & Столбец 2 & Столбец 3 \\
    \hline
    Данные    & Данные    & Данные    \\
    \hline
    \end{tabular}
\end{table}
```

### Формулы

```latex
Функция потерь задаётся выражением~\eqref{eq:loss}:
\begin{equation}
    L = \frac{1}{n} \sum_{i=1}^{n}(y_i - \hat{y}_i)^2,
    \label{eq:loss}
\end{equation}
где $y_i$ --- истинное значение, $\hat{y}_i$ --- предсказанное значение, $n$ --- объём выборки.
```

Пояснения символов начинаются словом «где» **без двоеточия** (п. 4.7.2).

### Списки

**Маркированный (тире):**
```latex
Система включает следующие модули:
\begin{itemize}
  \item удалённый заказ,
  \item виртуальная справочная служба,
  \item виртуальный читальный зал.
\end{itemize}
```

**Нумерованный (цифра со скобкой):**
```latex
Этапы работы:
\begin{enumerate}
  \item первичный осмотр материалов,
  \item сканирование документов,
  \item обработка полученных образов.
\end{enumerate}
```

### Библиография и ссылки на источники

Ссылки в тексте оформляются в **квадратных скобках** (п. 4.10.4):
```latex
Метод описан в работе [1].
Подробнее см. [2, с. 15].
```

Источники добавляются в `bibliography.bib`. Примеры записей:

```bibtex
% Статья в журнале
@article{ivanov2023,
  author  = {Иванов, И. И. and Петров, П. П.},
  title   = {Название статьи},
  journal = {Название журнала},
  year    = {2023},
  volume  = {1},
  number  = {2},
  pages   = {10--25}
}

% Книга / монография
@book{smith2022,
  author    = {Smith, J.},
  title     = {Book Title},
  publisher = {MIT Press},
  address   = {Cambridge},
  year      = {2022}
}

% Электронный ресурс
@misc{site2024,
  title        = {Название ресурса},
  howpublished = {URL: https://example.com},
  note         = {дата обращения: 01.03.2025}
}
```

Требования к оформлению библиографии (п. 4.10):
- Список размещается в конце основного текста ВКР, до приложений (п. 4.10.2)
- Источники нумеруются арабскими цифрами в порядке первого упоминания в тексте (п. 4.10.3)
- Ссылки в тексте — в квадратных скобках после цитаты или в конце предложения (п. 4.10.4)
- Записи оформляются по ГОСТ 7.11-2004, ГОСТ Р 7.0.5-2008, ГОСТ Р 7.0.100-2018 (п. 4.10.5)

### Список сокращений и терминов

Записи ведите в **алфавитном порядке**, **без знаков препинания в конце строки** (п. 4.8.4, 4.9.2):

```latex
% abbreviations.tex — сокращения без жирного шрифта, в алфавитном порядке
ВКР --- выпускная квалификационная работа\\[0.5em]
ГИА --- государственная итоговая аттестация
```

```latex
% terms.tex — термины без жирного шрифта, в алфавитном порядке
Классификация --- процесс распределения объектов по категориям\\[0.5em]
Нейронная сеть --- математическая модель, имитирующая работу мозга
```

### Добавление приложений

1. Создайте файл `appB.tex`, начните с `\chapter{НАЗВАНИЕ}`
2. Добавьте в `main.tex` после `\appendix`:
```latex
\input{appB.tex}
```

Буквы (А, Б, В, ...) присваиваются автоматически.

## Частые вопросы

**Q: Почему нет титульного листа?**
A: Титульный лист, задание и аннотация формируются в ИСУ ИТМО и прикрепляются к PDF отдельно.

**Q: Как изменить правое поле?**
A: В `preamble.tex` измените `right=15mm` на `right=10mm` — оба значения допускаются.

**Q: Почему не компилируется библиография?**
A: Убедитесь, что используете **Biber**, а не BibTeX. В Overleaf: Menu → Compiler → XeLaTeX (Biber подключается автоматически). Локально: запустите `biber main` вручную или через `latexmk`.

**Q: Как добавить список иллюстративного материала?**
A: Добавьте в `main.tex` перед библиографией:
```latex
\addcontentsline{toc}{chapter}{СПИСОК ИЛЛЮСТРАТИВНОГО МАТЕРИАЛА}
\listoffigures
```
