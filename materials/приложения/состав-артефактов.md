# Приложение Б. Состав экспериментальных артефактов

В таблице Б.1 приведён состав закоммиченных артефактов, на которых получены численные результаты главы 4. Артефакты сгруппированы по назначению и хранятся в каталоге `experiments/results/` репозитория проекта.

**Таблица Б.1.** Состав экспериментальных артефактов

| Группа | Назначение | Артефакты |
|---|---|---|
| Таблица единичных прогонов | Long-format таблица всех 486 единичных итераций параметрического симулятора с явным указанием значений всех параметрических осей конфигурации, идентификатора политики, номера реплики и значений показателей групп, описанных в подразделе 2.2.3 | `lhs_parametric_mobius_2025_autumn_2026-05-08.{json,csv,md}` |
| Файлы постобработки параметрического прогона | Сводки попарных сравнений, агрегатов по уровням параметрических осей, диагностики risk-positive подмножества, чувствительности по осям, разброса между репликами, диагностики политики на основе языковой модели и распределения «риск × релевантность» | `analysis_pairwise.json`, `analysis_capacity_audit.json`, `analysis_program_effect.json`, `analysis_gossip_effect.json`, `analysis_sensitivity.json`, `analysis_stability.json`, `analysis_llm_ranker_diagnostic.json`, `analysis_risk_utility.json` |
| Артефакты LLM-агентского симулятора | Long-format таблица 48 единичных итераций LLM-агентского симулятора на 12 точках репрезентативного подмножества | `llm_agents_lhs_subset_12pts.{json,csv,md}` |
| Артефакты перекрёстной проверки | Сводка коэффициентов ранговой корреляции Спирмена и совпадений верхнего лидера политик между параметрическим и LLM-агентским симуляторами на 12 общих точках, с пометками вырождения по точкам | `analysis_cross_validation.{json,md}` |
| Сводные отчёты | Компактные технические отчёты по результатам параметрического прогона (стадия U) и совместного прогона с перекрёстной проверкой (стадия W); содержат сводные таблицы и приёмочные чек-листы | `report_mobius_2025_autumn_parametric.md`, `report_mobius_2025_autumn_full.md` |
| Визуализации | Графические сводки результатов: распределение «риск × релевантность» в виде диаграммы рассеяния, средний риск по бакетам веса социального заражения в виде столбчатой диаграммы, тепловая карта ранжирования политик на 12 maximin-точках, согласованность параметрического и LLM-агентского симуляторов по метрикам в виде диаграммы коэффициентов корреляции Спирмена | `plots/analysis_risk_utility_scatter.png`, `plots/analysis_gossip_bucket_bar.png`, `plots/analysis_ranking_heatmap_maximin.png`, `plots/cross_validation_rho_per_metric.png` |
| Метаданные LLM-прогонов | Параметры запуска агентского симулятора, конкретные имена использованных языковых моделей (`openai/gpt-4o-mini` для политики на основе языковой модели, `openai/gpt-5.4-nano` для агентского симулятора), фактические значения суммарной стоимости и времени выполнения, число вызовов и доля ошибок парсинга структурированного ответа | Поля `params` и блок acceptance в `llm_agents_lhs_subset_12pts.json` |
| Тесты граничной верификации | Автоматизированные тесты, реализующие граничные свойства EC1–EC4 и их расширения на конфигурации с ненулевым весом социального заражения; каждый тест выполняется на минимально достаточной конфигурации с усреднением по нескольким случайным зёрнам | `experiments/tests/test_extreme_conditions.py` |

Сохранённые артефакты позволяют воспроизвести любую единичную итерацию параметрического прогона по фиксированному значению параметрических осей и случайных зёрен. Воспроизводимость LLM-агентских прогонов ограничена стохастичностью внешней языковой модели и фиксируется на уровне условий запуска.
