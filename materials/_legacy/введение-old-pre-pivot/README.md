# Введение, дофреймовая версия (архив)

Дата архивации: 2026-05-06.

Файл `введение.md` — черновик введения, написанный ДО пивота к финальному фрейму ВКР. Под старым фреймом работа формулировалась как «coordinated capacity-aware recsys для участников конференции» с центральным методическим ходом совместной оптимизации рекомендаций между пользователями. В качестве ближайшей литературной опоры цитировался ReCon (in-processing optimal transport).

Текущий фрейм (см. `<repo>/PROJECT_DESIGN.md` и `<repo>/PROJECT_STATUS.md`):
- задача формулируется как сценарная оценка риска перегрузки залов средствами системы поддержки принятия решений (СППР, Decision Support System);
- центральный методический ход — два независимых симулятора отклика для перекрёстной проверки выводов;
- объект исследования — процесс формирования программы конференции с точки зрения организатора, не процесс формирования персональных программ участников;
- от прогноза посещаемости и от accuracy-метрик отказались осознанно.

Файл хранится как свидетельство предыдущего этапа работы. Содержательно он не используется при подготовке нового введения: новый текст пишется с нуля под текущий фрейм, опираясь на `<repo>/PROJECT_DESIGN.md`, `<repo>/THESIS_ASSIGNMENT.md`, главы 1 и 2.
