# Глава 2, дофреймовая версия (архив)

Дата архивации: 2026-05-05.

Файл `черновик-главы-2.md` — черновик главы 2, написанный ДО пивота к финальному фрейму ВКР. Под старый фрейм работа формулировалась как «coordinated capacity-aware recsys для участников конференции» с центральным методическим ходом Constrained MDP плюс congestion games и обучением политики на синтетических взаимодействиях.

Текущий фрейм (см. `<repo>/PROJECT_DESIGN.md` и `<repo>/PROJECT_STATUS.md`):
- задача формулируется как сценарная оценка риска перегрузки залов средствами системы поддержки принятия решений (СППР, Decision Support System);
- центральный методический ход — два независимых симулятора отклика (параметрический и LLM-агентский) для перекрёстной проверки выводов;
- добавлена параметрическая ось локальных модификаций программы;
- от прогноза посещаемости и от accuracy-метрик отказались осознанно.

Файл хранится как свидетельство предыдущего этапа работы. Содержательно он не используется при подготовке новой главы 2: новая глава пишется с нуля под текущий фрейм, опираясь на `<repo>/PROJECT_DESIGN.md` и `<repo>/.claude/memory/research_field_survey_2026-05-04.md`.
