# Магистерская ВКР: рекомендательная система для конференций

Магистерская работа в ИТМО, Industrial AI, защита 2026.
Тема: интеллектуальная система поддержки формирования программы конференции с учётом ограничений вместимости параллельных сессий и согласованности рекомендаций между участниками.

Реальных данных JUG нет, поэтому всё исследование построено на симуляторе и сравнении политик в нём.

## Что в репозитории

```
masters-degree/
├── experiments/        — Python-код симулятора, политик, скриптов и результатов
├── thesis/             — LaTeX-исходник ВКР
├── materials/          — все рабочие материалы: черновики глав, заметки, переписка
├── external/           — клонированные сторонние репозитории (например, MiroFish для разбора)
└── README.md           — этот файл
```

Сосед: `../jug-rec-sys` — открытый проект коллектива научного руководителя
с реальной инфраструктурой (Qdrant, REST-сервис рекомендаций, телеграм-бот).
Наша работа описывает себя как надстройку над этой системой.

### Главные числа защиты

- Capacity-aware MMR в 2.2× меньше переполнений, чем косинусная политика, на симуляторе с 1200 участниками и реалистичной вместимостью залов.
- При высокой восприимчивости к подсказке (compliance = 0.9): в 55× меньше.
- Cross-domain на MovieLens 1M: относительный порядок политик сохраняется (Spearman ρ = +0.77, p = 0.016 для среднего избытка над вместимостью).
- Stylized facts: 2 из 3 воспроизводятся (Парето + спад посещаемости от слота к слоту).

## Соглашения

- Язык текста ВКР — русский, академический стиль, безличные конструкции, минимум англицизмов.
- Все рабочие черновики глав — в `materials/глава-X/черновик-главы-X.md`. Сначала пишем туда, потом синхронизируем с `thesis/chapter*.tex`.
- Виртуальное окружение Python поднимается через `uv` (`uv venv`, `uv pip install ...`).
- Сборка PDF: `cd thesis && latexmk -xelatex main.tex`.
- Не ставить пакеты глобально без явного разрешения.
