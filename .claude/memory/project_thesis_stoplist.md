Стоп-лист тезисов, которые больше **НЕ** защищаются в ВКР (зафиксирован 2026-05-04).

**Why:** проект пережил несколько пивотов; ряд старых утверждений противоречит финальному фрейму (см. `project_final_thesis_frame.md`) или содержит методологические натяжки.

Запрещённые формулировки:

1. «Лучший персональный рекомендатель докладов».
2. «Cap-aware MMR — победитель» — опровергнуто E2.
3. «Предсказание реальной посещаемости конференции» — нет real attendance data.
4. «B1 / accuracy@1 = 0.918 как внешняя валидация preference-модели» — leakage через memberships (RSVP только в свои группы).
5. «Cross-domain Spearman как валидация против реальности» — это самосогласованность четырёх симуляторов, не валидация.
6. «Big Five / social graph / inter-slot chat как реализованный метод» — это прототипы из старых пивотов, удалены из `experiments/src/agents/`.
7. «CoSPLib как attendance / preference / capacity benchmark» — там нет ни attendance, ни capacity, ни preferences, ни talk text. Использовать только cite-only как conference scheduling benchmark.
8. «UMass — конференция» — это course allocation, смежный домен limited-resource allocation.
9. «Симулятор реалистично воспроизводит конференцию» — нет калибровки на real attendance.
10. «LLM distribution-match ρ=0.438 = realism симулятора» — это face validity / believability, не validity (Larooij & Törnberg 2025).

**How to apply:**
- При написании текста, аннотации, презентации — проверять, что ни одна из этих формулировок не появляется явно или скрыто.
- При вопросах от пользователя — поправлять, если он начинает использовать стоп-формулировку.
- При появлении новых тезисов — добавлять сюда, если они проваливают верификацию.
