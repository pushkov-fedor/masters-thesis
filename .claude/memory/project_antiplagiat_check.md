**Дедлайн:** до 8 мая 2026 включительно (открыто с 4 мая 2026).

**Куда:** почта `aitalents@itmo.ru`, тема письма «Проверка ВКР в системе Антиплагиат». Формат PDF, имя файла `ИвановИИ_ВКР` (под фамилию студента).

**Проверка обязательная для всех треков, кроме «бизнес-тезиса»** — там проверяет секретарь ОГЭК трека самостоятельно.

**Требование оригинальности:** более 80%.

**Формат для предзащиты — вручную добавить в начало файла (это временный формат, в финальной загрузке появится автоматически):**
1. ФИО студента
2. Тема ВКР
3. Задание на ВКР — раздел «Основные вопросы»
4. Аннотация — цель, задачи, краткие результаты
5. Далее обычный текст работы. Содержание = первая страница нумерации.

**Why:** обязательная веха перед предзащитой; невыполнение блокирует дальнейшие шаги.

**How to apply:** держать в голове при планировании работы по тексту ВКР до начала мая 2026; перед сборкой PDF не забыть про вручную добавляемый префикс (ФИО → тема → задание → аннотация → содержание). На основную работу остаётся ~неделя с момента получения дедлайна (29.04 → 08.05).

**Связано:** аннотация для предзащиты — структура и примеры в `materials/Аннотация ВКР — требования.md` (цель, задачи, результаты; заголовки не дублировать; на английском, если защита на английском; для попадания публикаций — заполнить Портфолио в ИСУ).
